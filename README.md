**What it do? :o**
If you're here, you already know the pain. The default Windows 11 auto-hide taskbar is completely broken. It's buggy, it gets stuck, it stutters, or it just snaps away instantly with zero animation. It's ridiculously frustrating when you just want a clean, dark desktop setup without the UI glitching out and ruining the vibe.

I got tired of it not working right and didn't want to install some bloated third-party software just to fix something Microsoft should have gotten right in the first place.
  
  
**What it be**
This is a stupidly simple, single-click fix. It’s just a lightweight .bat file. You double-click it, and it forces the taskbar to do a buttery-smooth slide down off your screen using Windows' own native code. No jank, no stuttering.
  
  
  
**How to use it**
Download the SmoothTaskbar.bat file.

Throw it on your desktop (or wherever is easiest).

Just **double-click it**.

A black command window will flash for a split second, and your taskbar will smoothly slide down and out of view. If you want it back and locked in place, just double-click the file again

That's literally it. No installation, no background memory hogging, just a working taskbar animation. Enjoy!
